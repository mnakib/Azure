
# AZ‑104 Custom Course Outline

## Description
The **custom AZ‑104: Microsoft Azure Administrator** training is a structured course to give you flexibility—ideal for freelance delivery, blended learning, or adapting to a client’s environment.

## Duration
4 days

## Outline

This outline covers all core AZ‑104 exam domains while giving room to tailor labs, demos, and depth based on audience experience.

---

## **Module 1 — Azure Identity & Governance**
### Learning Objectives
Build a solid foundation in Azure AD, RBAC, and governance controls.

### Topics
- Azure AD vs. Entra ID concepts  
- Users, groups, and administrative roles  
- Azure AD Join, device identities, hybrid identity overview  
- Role-Based Access Control (RBAC)  
- Azure subscriptions & management groups  
- Governance tools: Azure Policy, Blueprints, Tags  
- Cost management fundamentals  

### Labs
- Create users & groups  
- Assign RBAC roles  
- Apply Azure Policy to enforce standards  

---

## **Module 2 — Azure Compute Administration**
### Learning Objectives
Deploy and manage compute workloads efficiently.

### Topics
- Virtual machine creation, sizing, and availability options  
- VM storage, extensions, and custom script automation  
- VM networking (NICs, IPs, NSGs)  
- Azure App Services overview  
- Azure Container Instances & basic container management  
- Azure Virtual Desktop (optional add‑on)  

### Labs
- Deploy a VM with custom configuration  
- Configure VM backup & monitoring  
- Deploy a simple web app to App Service  

---

## **Module 3 — Azure Storage Management**
### Learning Objectives

Understand and manage Azure storage services securely and efficiently.

### Topics
- Storage accounts, redundancy (LRS/ZRS/GZRS)  
- Blob storage, containers, lifecycle management  
- File shares (SMB/NFS)  
- Azure Storage Explorer  
- Shared Access Signatures (SAS) & access keys  
- Azure Backup & Recovery Services Vault  

### Labs
- Create a storage account with secure access  
- Configure lifecycle rules  
- Implement Azure Backup for VMs or files  

---

## **Module 4 — Azure Networking**
### Learning Objectives
Build and manage secure, scalable network architectures.

### Topics
- Virtual networks, subnets, and IP addressing  
- Network Security Groups (NSGs) & Application Security Groups  
- Azure Firewall & network appliances  
- VPN Gateway & ExpressRoute fundamentals  
- Load balancers (public, internal)  
- Private Endpoints & service endpoints  
- DNS zones & name resolution  

### Labs
- Build a hub‑and‑spoke network  
- Configure a load balancer  
- Deploy a site‑to‑site VPN (simulated lab)  

---

## **Module 5 — Monitoring, Automation & Resilience**
### Learning Objectives
Ensure workloads are monitored, automated, and resilient.

### Topics
- Azure Monitor, Log Analytics, and metrics  
- Alerts & action groups  
- Azure Automation (Runbooks, Update Management)  
- Azure Resource Manager (ARM) templates & Bicep basics  
- High availability & disaster recovery strategies  
- Azure Advisor & best‑practice recommendations  

### Labs
- Configure monitoring and alerts  
- Deploy resources using ARM/Bicep  
- Automate VM updates  

---
